# Rubiks Cube 

Simulated Rubiks Cube where you can rotate faces using the keys(F,B,U,D,L,R) and scroll around to see the cube fully.

## Controls

Rubik's Cube Simulation Controls:  

Click and Drag Left Mouse button to adjust Camera  

Press Shift to reverse direction of rotation  
F : Rotate Front Face (clockwise)  
B : Rotate Back Face (clockwise)  
U : Rotate Up Face (clockwise)  
D : Rotate Down Face (clockwise)  
R : Rotate Right Face (clockwise)  
L : Rotate Left Face (clockwise)  

ESC : Quit simulation   

## How to Run

Run the binary `./rubik`, if it doesnt exists just run the Makefile and it should create it.
